# SAtronix Technology

Premium digital agency website built with React, Vite, Tailwind CSS, Framer Motion, and React Three Fiber.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Three Fiber / Three.js
- React Router
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
